12.1.14. Physical parameters¶

12.1.14.1. GRAVITY¶

Data Type: DOUBLE PRECISION

Gravitational acceleration. [9.807 m/s^2 in SI]

12.1.14.2. GRAVITY_X¶

Data Type: DOUBLE PRECISION

X-axial component of gravitational acceleration vector.

12.1.14.3. GRAVITY_Y¶

Data Type: DOUBLE PRECISION

Y-axial component of gravitational acceleration vector.

12.1.14.4. GRAVITY_Z¶

Data Type: DOUBLE PRECISION

Z-axial component of gravitational acceleration vector.

12.1.14.5. P_REF¶

Data Type: DOUBLE PRECISION

Reference pressure. [0.0]

12.1.14.6. P_SCALE¶

Data Type: DOUBLE PRECISION

Scale factor for pressure. [1.0]
